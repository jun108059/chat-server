## 요구사항

1. 회원 시스템
   - 개인 / 기업 / 관리자
2. 관리자가 채팅방 생성
   - 기업 기반 채팅방 생성이 가능해야 함
3. 개인 회원(N) : 기업(1) 채팅 참여
   - 실시간 채팅 가능

### Task

1. 이벤트 스토밍
2. 도메인 정의
3. UseCase 도출
4. Test code 작성(TDD)
5. API 서버 개발
6. Socket 서버 개발
7. Kafka 파이프라인 구성
